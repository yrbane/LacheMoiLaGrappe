Politique de Confidentialité - LacheMoiLaGrappe

Dernière mise à jour : 6 mars 2026

Introduction

LacheMoiLaGrappe est une application de filtrage d'appels qui respecte votre vie privée. Cette politique explique comment l'application traite vos données.

Données collectées

LacheMoiLaGrappe ne collecte aucune donnée personnelle et ne transmet aucune information à des serveurs externes. Toutes les données restent stockées localement sur votre appareil :

  • Historique des appels filtrés : numéros de téléphone, date, décision de filtrage
  • Préférences utilisateur : paramètres de filtrage, template SMS
  • Listes utilisateur : numéros autorisés et bloqués
  • Historique des SMS envoyés : si la fonctionnalité SMS est activée
  • SMS suspects détectés : si la détection phishing est activée (numéro, contenu, mot-clé détecté)

Permissions utilisées

  • READ_CONTACTS : pour identifier les appelants connus et laisser passer leurs appels
  • READ_PHONE_STATE / READ_CALL_LOG : pour filtrer les appels entrants
  • SEND_SMS (optionnel) : pour envoyer un SMS de réponse automatique aux appelants inconnus, uniquement si vous activez cette fonctionnalité
  • RECEIVE_SMS (optionnel) : pour analyser les SMS entrants et détecter les tentatives de phishing, uniquement si vous activez cette fonctionnalité
  • POST_NOTIFICATIONS : pour vous informer des appels rejetés et des SMS suspects

Partage de données

LacheMoiLaGrappe ne partage aucune donnée avec des tiers. Il n'y a aucun SDK d'analyse, de publicité ou de suivi intégré dans l'application.

Stockage et sécurité

Toutes les données sont stockées dans le stockage privé de l'application sur votre appareil, dans une base de données chiffrée avec SQLCipher (AES-256). La clé de chiffrement est protégée par le Android Keystore matériel. Les données sont supprimées automatiquement si vous désinstallez l'application.

Vos droits

Vous pouvez à tout moment :

  • Consulter l'historique des appels filtrés dans l'application
  • Supprimer toutes les données en désinstallant l'application
  • Révoquer les permissions via les paramètres Android

Modifications

Cette politique peut être mise à jour. Les changements seront publiés sur cette page.
